Add a type and query function:

```typescript
export type MonthlySpend = {
  month: string    // "YYYY-MM" format
  label: string    // "Jan", "Feb", etc. for chart display
  totalCents: number
}
```

Export async function `fetchBrokerMonthlySpend(brokerId: string, months = 12): Promise<MonthlySpend[]>`:

1. Query orders table for this broker where total_price_cents is not null, selecting: `total_price_cents, created_at`
2. Filter to orders created within the last N months: calculate the cutoff date as `new Date()` minus `months` months, use `.gte('created_at', cutoff.toISOString())`
3. Group in JavaScript (not SQL since Supabase doesn't support GROUP BY easily via the JS client):
   - Create a Map<string, number> keyed by "YYYY-MM"
   - For each order, extract the month from created_at and add total_price_cents to the running sum
4. Generate the full month range (even months with zero spend) by iterating from the cutoff month to current month
5. For each month in range, push { month: "YYYY-MM", label: short month name (e.g. "Jan '25"), totalCents: sum or 0 }
6. Return sorted chronologically (oldest first, for left-to-right chart reading)

Use the short month format for labels: `new Date(year, monthIndex).toLocaleString('en-US', { month: 'short' })` plus the 2-digit year, like "Jan '25".
